| Тип             | Назначение                                                     | Пример                                                                                  |
| :-------------- | :------------------------------------------------------------- | :-------------------------------------------------------------------------------------- |
| **`feat:`**     | Новая функциональность *(feature)*                             | `feat: добавлен чат для пользователей`<br>`feat: added chat for users`                  |
| **`fix:`**      | Исправление бага *(bug fix)*                                   | `fix: ошибка при сохранении рецепта`<br>`fix: bug when saving recipe`                   |
| **`docs:`**     | Изменения только в документации *(documentation)*              | `docs: обновлено README`<br>`docs: updated README`                                      |
| **`style:`**    | Правки, не влияющие на логику *(code style, formatting)*       | `style: привёл код к eslint правилам`<br>`style: aligned code with eslint rules`        |
| **`refactor:`** | Переработка кода без изменения поведения *(code refactoring)*  | `refactor: упрощена логика фильтрации`<br>`refactor: simplified filtering logic`        |
| **`perf:`**     | Улучшение производительности *(performance)*                   | `perf: оптимизирован запрос к БД`<br>`perf: optimized DB query`                         |
| **`test:`**     | Добавление или изменение тестов *(tests)*                      | `test: добавлены юнит-тесты для API`<br>`test: added unit tests for API`                |
| **`chore:`**    | Прочие задачи *(dependencies, configs, maintenance)*           | `chore: обновлён eslint`<br>`chore: updated eslint`                                     |
| **`build:`**    | Изменения, влияющие на сборку или зависимости *(build system)* | `build: добавлен webpack-плагин`<br>`build: added webpack plugin`                       |
| **`ci:`**       | Изменения в CI/CD конфигурации *(continuous integration)*      | `ci: добавлен запуск тестов в GitHub Actions`<br>`ci: added test run in GitHub Actions` |
